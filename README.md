# Proyecto-6_Enrutamiento_IPRoute
Enrutamiento IP Router o Estático.
Copia y Pega cada contenido de los Routers en la plantilla sin configurar.
Por último debes configurar cada PC con sus datos de red, ejemplo: IP: 192.168.1.2 / Submáscara: 255.255.255.0 / Gateway: 192.168.1.1
Recuerda que el Router1 debes añadirle 1 módulo WIC-2T (para conectarse a ambos lados de la red).
